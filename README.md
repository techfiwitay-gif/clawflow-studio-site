# OpenClaw Setup Website (MVP)

## Files
- `index.html`
- `styles.css`

## Run locally
```bash
cd openclaw-setup-site
python3 -m http.server 8081
```
Then open: `http://localhost:8081`

## Quick customize
- Brand name: currently set to `ClawFlow Studio`
- Booking link: currently set to `https://calendly.com/techfiwitay`
- Email: currently set to `techfiwitay@gmail.com`
- Colors: set to white/milky theme in `styles.css`
- Prices/packages: edit the `#packages` section

## Deploy options
- Netlify: drag-and-drop this folder
- Vercel: import as static site
- GitHub Pages: push folder to repo and enable Pages
